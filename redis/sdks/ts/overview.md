`@upstash/redis` is an HTTP/REST based Redis client for TypeScript, built on top of Upstash REST API. You can find the Github Repository here. It is the only connectionless (HTTP based) Redis client and designed for:
  * Serverless functions (AWS Lambda …)
  * Cloudflare Workers (see the example)
  * Fastly Compute@Edge (see the example)
  * Next.js, Jamstack …
  * Client side web/mobile applications
  * WebAssembly
  * and other environments where HTTP is preferred over TCP.

See the list of APIs supported.
