`upstash-redis` is a connectionless, HTTP-based Redis client for Python, designed to be used in serverless and serverful environments such as:
  * AWS Lambda
  * Vercel Serverless
  * Google Cloud Functions
  * and other environments where HTTP is preferred over TCP.

Inspired by other Redis clients like @upstash/redis and redis-py, the goal of this SDK is to provide a simple way to use Redis over the Upstash REST API. The SDK is currently compatible with Python 3.8 and above. You can find the Github Repository here.
